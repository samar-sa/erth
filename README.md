# Erth - إرث

Erth is a digital platform dedicated to documenting archaeological and mineral wealth in the Kingdom of Saudi Arabia. The platform transforms traditional data into a secure and organized digital record, making it easily accessible and contributing to the national digital transformation.

# Features
Full Data Management (CRUD): Easily add, view, update, and delete records of artifacts, archaeological sites, and mineral deposits.
Documentation and Archival: Support for uploading images, detailed descriptions, types, and categories for each discovered asset.
Secure Architecture: Built utilizing Laravel standards to ensure data integrity and protected access control.
Clean UI: A simple and intuitive dashboard designed for smooth navigation and fast data filtering.

# Tech Stack
Backend Framework: Laravel (PHP)
Database Management: MariaDB
Frontend: Blade Templates, HTML5, CSS
Version Control: Git and GitHub
